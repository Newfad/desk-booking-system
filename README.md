# Desk Booking System

A web application built with Python and Flask for Northumbria University staff to book desks.

## Features
- User login and registration
- View and book available desks
- Manage bookings
- Admin portal for managing desks and users

## Installation
1. Clone the repository
2. Install dependencies: `pip install -r requirements.txt`
3. Run the app: `python app.py`
